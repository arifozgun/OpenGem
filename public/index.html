Chat Completions

POST /v1beta/models/{model}:generateContent

Request Body

Parameter Type Required Description
contents array Yes Array of content objects with role and parts
systemInstruction object No Sets the behavior and instructions for the model
generationConfig object No Configuration options like temperature and maxOutputTokens

Content Parts

Field Description
text The string text of the prompt
inlineData For multimodal inputs (e.g. base64 image data and mimeType)

Code Examples

curl -X POST "http://localhost:3050/v1beta/models/gemini-3.1-pro-preview:generateContent?key=sk-your-api-key" \
  -H "Content-Type: application/json" \
  -d '{
    "contents": [{
      "parts": [{"text": "Hello!"}]
    }]
  }'
pip install google-genai
from google import genai

client = genai.Client(
    api_key="sk-your-api-key",
    http_options={'api_version': 'v1beta', 'url': 'http://localhost:3050'}
)

response = client.models.generate_content(
    model="gemini-3.1-pro-preview",
    contents="Hello!",
)

print(response.text)
npm install @google/genai
import { GoogleGenAI } from '@google/genai';

const ai = new GoogleGenAI({
  apiKey: 'sk-your-api-key',
  baseUrl: 'http://localhost:3050',
});

const response = await ai.models.generateContent({
  model: 'gemini-3.1-pro-preview',
  contents: 'Hello!'
});

console.log(response.text);
pip install langchain-google-genai
from langchain_google_genai import ChatGoogleGenerativeAI

llm = ChatGoogleGenerativeAI(
    google_api_key="sk-your-api-key",
    client_options={"client_cert_source": None, "api_endpoint": "http://localhost:3050"},
    model="gemini-3.1-pro-preview"
)

response = llm.invoke("Hello!")
print(response.content)
